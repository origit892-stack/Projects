# Roblox Studio Essential Shortcuts

## Navigation
| Shortcut | Action |
|----------|--------|
| WASD | Move camera in viewport |
| Mouse scroll | Zoom in/out |
| Right-click drag | Rotate camera |
| Middle-click drag | Pan camera |
| F | Focus selected part |
| Shift+F | Frame selected in view |
| Ctrl+Shift+F | Frame all |

## Plugin: Rojo
- Install via Plugin Manager
- Click Rojo icon in toolbar
- Enter server address (default: `localhost:34872`)
- Click **Connect** then **Sync**
- Enable **Live Sync** for auto-updates

## Tips
1. **Use "Anchor" on all static parts** — unanchored parts fall/respawn
2. **Name everything** — makes scripting and debugging 10x easier
3. **Group related parts** — organize with Models
4. **Use the Properties window** — faster than scripting common changes
5. **Save frequently** — Ctrl+S
6. **Use "Publish to Roblox"** — keeps cloud backup
