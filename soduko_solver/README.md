# Sudoku Atelier

פותר סודוקו מלא עם ממשק עברי רספונסיבי.

## הפעלה

אפשר לפתוח את `index.html` ישירות בדפדפן, או להריץ שרת מקומי:

```bash
python3 -m http.server 8080
```

לאחר מכן פותחים את `http://localhost:8080`.

## יכולות

- פתרון סודוקו באמצעות backtracking עם בחירת התא המצומצם ביותר
- זיהוי התנגשויות בזמן אמת
- תמיכה מלאה במקלדת ובמובייל
- הדבקת לוח שלם כ־81 תווים
- ביטול פעולה, ניקוי וחידה לדוגמה
- אנימציית פתרון וחגיגת הצלחה
