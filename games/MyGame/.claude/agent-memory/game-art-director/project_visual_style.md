Style name: "Dark Neon Dungeon" — proposed March 2026, pending developer approval.

Core philosophy: Deep void backgrounds, neon accent lighting that bleeds and blooms, dramatic contrast between darkness and color. Every color choice communicates gameplay state.

**Why:** Game already has the right bones — dark backgrounds, purple/violet player, glowing portals. Proposal elevates existing direction rather than replacing it.

**How to apply:** Treat this as the target palette for any future color or sprite discussions. If developer approves elements, mark them as confirmed.
