Claude Code is an AI coding assistant from Anthropic that works directly in your terminal and IDE. It understands full codebases, writes and edits files, runs tests, and helps ship complex software through natural conversation.

Not just autocomplete — a collaborative engineer that thinks through architecture, spots bugs before you do, and ships working code in one session. Every game here was built start-to-finish in a single conversation.

How It Was Built

01 · PROCESS
Single Conversation

Each game was built and refined through back-and-forth with Claude Code — iterating on mechanics, fixing bugs in real time, and tuning the feel of every system from physics to procedural Web Audio synthesis.

02 · ARCHITECTURE
Zero Dependencies

Every game is a single .html file you can open directly in any browser. No npm, no bundler, no server required. The entire game loop, audio engine, renderer, and UI live in one file.

03 · OUTPUT
Ship-Ready

From concept to playable in one session. Claude Code handles architecture, writes the game loop, implements audio, polishes UX — all while explaining every decision and catching edge cases before you hit them.
