# רכבת 13

משחק אימה פסיכולוגי תלת־ממדי שמתרחש ברכבת לילה אינסופית. הקרון בנוי בזמן אמת ב־WebGL עם Three.js מקומי, חומרים פרוצדורליים, תאורה דינמית ואובייקטים אינטראקטיביים. בכל קרון צריך לזהות אם קיימת חריגה:

- אם הכול תקין — עוברים בדלת לקרון הבא.
- אם משהו השתנה — מושכים בידית החירום.
- החלטה נכונה מקדמת את הנוסע לעבר קרון 13.
- החלטה שגויה מחזירה את הנוסע לאחור.

## הפעלה

בגלל השימוש במודולי JavaScript יש להפעיל שרת מקומי:

```bash
cd /Users/origan/Documents/Projects/train_13
python3 -m http.server 8080
```

ואז לפתוח `http://localhost:8080`.

## שליטה

- עכבר או מגע — להביט סביב הקרון
- לחיצה על הדלת / `E` / `Enter` — הקרון תקין
- לחיצה על ידית החירום / `R` — נמצאה חריגה
- `M` — השתקת קול
- `F` — מסך מלא

Three.js שמור מקומית בתיקיית `vendor`, ולכן המשחק אינו תלוי ב־CDN. כל הטקסטורות נוצרות בזמן אמת וכל הסאונד נוצר דרך Web Audio API. בדפדפן ללא WebGL המשחק חוזר אוטומטית לגרסת ה־CSS המקורית.
